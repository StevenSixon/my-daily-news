## 它是什么
CubeSandbox 是腾讯云开源的高性能安全沙箱服务，基于 RustVMM 和 KVM 为 AI Agent 提供即时、并发的硬件级隔离执行环境。每个沙箱拥有独立 Guest OS 内核，避免 Docker 共享内核逃逸风险，冷启动低于 60 毫秒，内存开销低于 5MB，支持单节点数千实例高密度部署。原生兼容 E2B SDK，可零代码迁移现有 Agent 工作流。

## 为什么火
AI Agent 频繁执行不可信代码（如 LLM 生成的 Python/Shell），Docker 隔离性弱易逃逸，传统虚拟机启动慢（秒级）、资源占用高。CubeSandbox 在安全与性能间取得极致平衡：硬件级隔离 + 毫秒级启动 + 超低内存开销，并集成 E2B 生态，成为安全执行 AI 生成代码的理想基座。

## 技术栈
- 语言：Rust（核心虚拟化），Go（管理组件），Lua（出口代理）
- 虚拟化：RustVMM + KVM
- 网络代理：OpenResty/nginx + TPROXY 实现 L7 出口管控
- 管理：cubecli 命令、Web 控制台
- 模板：OCI 镜像一键转换
- 兼容性：E2B SDK，PyPI 包 cubesandbox

## 核心能力
- **毫秒级启动**：平均 <60ms，高并发下 P99 <150ms
- **硬件隔离**：独立内核，eBPF 增强安全
- **超低开销**：基础内存 <5MB，支持最高 32GB 沙箱规格
- **E2B 兼容**：替换环境变量即可迁移现有代码
- **快照/克隆/回滚**：基于 Copy-on-Write 的百毫秒级操作（v0.3 引入）
- **CubeEgress 安全代理**：凭证注入（密钥不进入沙箱）、域名过滤、访问审计（v0.4 引入）
- **Web 控制台**：模板管理、节点状态、版本矩阵一览
- **模板系统**：OCI 镜像制模板，模板商店预置，自动分发
- **高并发密度**：单节点可运行数千沙箱

## 适用场景
- AI Agent 代码解释器（替代 OpenAI Code Interpreter）
- 多租户安全代码执行平台
- 强化学习训练环境（如 SWE-bench 评估）
- 浏览器沙箱、主机挂载等自定义负载
- 数字助理/OpenClaw 集成

## 同类对比
| 维度 | Docker 容器 | 传统 VM | CubeSandbox |
|------|------------|--------|-------------|
| 隔离性 | 低（共享内核） | 高 | 极高（独立内核+eBPF） |
| 启动速度 | 200ms | 秒级 | <60ms |
| 内存开销 | 低 | 高 | 极低（<5MB） |
| 部署密度 | 高 | 低 | 极高 |
| Agent 生态 | 无 | 无 | E2B SDK 兼容 |

相比 Firecracker 等微 VM，CubeSandbox 在 Agent 集成度与 E2B 生态兼容上更具优势。
