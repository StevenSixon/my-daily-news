### 安装
```bash
brew install --cask fluidvoice
```
或从 [Releases](https://github.com/altic-dev/FluidVoice/releases/latest) 下载 DMG。

### 前置要求
- macOS 15.0 或更新
- Apple Silicon Mac（推荐，Intel 仅支持 Whisper 模型）
- 麦克风权限
- 辅助功能权限（用于打字插入）
- 约 1 GB 磁盘空间（基础语音模型）

### 最小可用步骤
1. 首次启动时，根据引导授予麦克风和辅助功能权限。
2. 选择一个语音模型：推荐从 Apple Speech（零下载）或 Parakeet Flash（英语低延迟）开始。
3. 设置一个全局快捷键（例如 `⌃⌥␣`）。
4. 按下快捷键开始说话，实时结果直接输入到当前活动文本框。
5. （可选）在「设置」中启用 Fluid Intelligence 或配置云 AI 提供方以获得格式化增强。
