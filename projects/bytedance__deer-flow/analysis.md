## 它是什么
DeerFlow 是由字节跳动开源的超级智能体框架（2.0 版，完全重写），它通过编排子代理、记忆系统、沙箱环境和可扩展技能，能够处理从几分钟到几小时的复杂任务。

## 为什么火
- GitHub 7.2万星，2026年2月登顶 Trending #1，社区高度活跃
- 支持多种 LLM 提供商（OpenAI、DeepSeek、vLLM 等），灵活配置
- 官方提供 Coding Plan 和 InfoQuest 爬虫工具集
- 强调长程任务执行与安全隔离

## 技术栈
- 后端: Python 3.12+，依赖 LangChain/LangGraph，支持 Claude Code、Codex CLI 等接口
- 前端/工具: Node.js 22+，TypeScript 辅助
- 部署: Docker 容器化，支持开发热重载和生产模式
- 配置: config.yaml + .env，交互式配置向导

## 核心能力
- **技能与工具**: 可扩展技能系统，集成 Claude Code 等外部编码代理
- **子代理**: 并行/层级的多代理协作
- **沙箱与文件系统**: 隔离执行环境，支持 AIO 沙箱供应商
- **上下文工程**: 管理长对话上下文，防止遗忘
- **长期记忆**: 持久化存储，跨会话保留知识
- **消息网关**: 统一的通信层（README 未展开细节）

## 适用场景
- 深度研究：生成报告、资料收集
- 自动化编码：多步骤开发、修复、测试
- 智能助手：结合沙箱和记忆的长期任务
- 多代理系统：需要协调子代理的复杂工作流

## 同类对比
相比 LangChain/LangGraph 基础框架，DeerFlow 提供更完整的开箱即用能力（沙箱、记忆、技能），但耦合度较高。与 AutoGPT、CrewAI 等相比，强调长程任务和字节内部产品集成。
