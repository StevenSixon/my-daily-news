## 它是什么

iLab GPT Conjure 是一个面向 GPT-image-2 模型的本地 AI 图像生成 WebUI 工作台。它通过 FastAPI 提供 Web 界面，并支持 CLI 调用，内置公用图库、提示词模板、多任务队列和历史管理。用户可配置 OpenAI 兼容 API 或本机 Codex OAuth 通道，实现文生图、参考图生成、图像编辑等流程。项目提供 Windows 和 macOS 的免安装一键包，解压即用。

## 为什么火

该项目在短时间内获得 544 星，主要因为它填补了 GPT-image-2 缺乏专门界面的空白。官方仅提供 API，而 iLab GPT Conjure 将调用、管理、复用、队列、编辑整体打包成本地工作台，大幅降低了使用门槛。免安装一键包的设计类似于 ComfyUI 下的便携体验，吸引了大量不想折腾环境的用户。多语言支持（含中文）和开放源码进一步推动了它在非英语社区的传播。

## 技术栈

- 后端：Python 3.11+、FastAPI、SQLite（历史库）、本地文件存储
- 前端：TypeScript、Konva（画布编辑器），静态资源构建后内嵌于 WebUI
- CLI：通过 `python -m codex_image` 提供生成、编辑等命令行功能
- API 接入：支持 OpenAI 兼容 API（Images API / Responses API），并可选本地 Codex OAuth 调用

## 核心能力

- **多认证模式**：推荐的 OpenAI 兼容 API 模式，适合团队和稳定服务；高级模式支持本机 Codex OAuth，用于个人实验。
- **提示词增强**：三种 chip（图库 chip、颜色 chip、提示词片段 chip）和模板系统，可快速组装复杂提示词，提升复用效率。
- **图像编辑**：内置多图层画布编辑器，支持插入参考图、缩放旋转、擦除，且默认锁定比例，适合精确调整输入图。
- **任务管理**：多任务并发、本地队列、分页历史库，可搜索、重试、打包下载。
- **公用图库**：本地资源库，支持分类、命名、拖拽排序，可通过 `@` 快捷引用。
- **多语言与便携**：支持 13 种语言，首次启动自动匹配浏览器语言；官方提供免安装 zip 包，覆盖 Windows x64、macOS Apple Silicon 和 Intel 三个平台。

## 适用场景

- 个人创作者使用 GPT-image-2 进行批量或灵感式图像生成
- 设计团队共享品牌素材图库，统一输出风格
- 开发者在本地集成 AI 图像生成并快速测试提示词模板
- 需要稳定、可记录的图像生成历史的工作流

## 同类对比

与 OpenAI 官方 Playground 相比，本工具提供了持久的历史记录、图库复用、模板管理和多任务队列，更适合重复性生产任务。与 ComfyUI 相比，它专注于 GPT-image-2 而非 Stable Diffusion 节点编排，面向更轻量的单模型调用工作流。但其依赖外部 API 的本质决定了生成速度受限于供应商，不算本地推理方案。

## 版本动态

最新版本 v0.5.2（2026-06-21）重点引入多语言界面和输入图像画布编辑。画布支持多图层组合、首图/自动扩展画布范围，图层缩略图实时显示。同时修复了任务状态同步和公用图库窄卡片溢出问题。免安装包覆盖三大平台，提供手动更新脚本。项目采用 AGPL-3.0 许可，要求网络服务修改需开放源码。
