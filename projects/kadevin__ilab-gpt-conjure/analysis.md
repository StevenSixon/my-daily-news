## 它是什么
iLab GPT Conjure 是为 GPT-image-2 及 OpenAI 兼容图像 API 打造的本地 WebUI 工作台（兼 CLI），提供文生图、图生图、图像编辑界面。它像 Stable Diffusion WebUI 一样操作，但调用的是远程模型，支持 Codex 内部通道和任何兼容供应商。内置公用图库、三种提示词芯片、模板库、多任务并发和历史管理，并提供 Windows/macOS 免安装一键包。

## 为什么火
- 填补了 GPT-image-2 无专业本地工作台的空白，让拒绝网页版限制的高级用户获得桌面级体验。
- 类似 ComfyUI 的“解压即用”便携包，零配置即可运行，降低上手门槛。
- 独创的 chip 系统（图库引用、颜色锁定、片段快捷填入）与模板库紧密集成，将复杂提示词工程模块化，大幅提高生图迭代速度。
- 完全本地运行的数据与界面，支持多供应商 API，避免被单一服务绑定。

## 技术栈
后端：Python 3.11+、FastAPI、Uvicorn、SQLite。前端：TypeScript、Konva（图层编辑）、预构建静态资源。支持 Windows x64、macOS ARM/Intel 平台，通过 CI/CD 自动打包。认证层同时支持本机 OAuth 提取和 API Key 注入。

## 核心能力
- **多接口集成**：Codex Image 通道、Codex Responses 通道、OpenAI 兼容 Images/Responses API，可在设置中自由切换。
- **图像工作流**：文生图、参考图生图、多图层编辑（锁定比例、局部擦除、遮罩）、批量导出、失败重试。
- **提示词芯片**：`@` 搜索公用图库并自动插入参考图；`#` 写入十六进制颜色约束；`~` 保存/展开常用提示词片段。
- **图库与模板**：公用图库可分类、命名、搜索；提示词模板支持缩略图预览、插入/替换、导入导出。
- **任务并发与历史**：本地队列管理，可设并发上限；/history 页面支持 SQLite 分页、搜索、网格/列表视图。
- **便携部署**：免安装 zip 包含打包的 CPython 和所有依赖，双击启动；更新脚本自动校验 SHA256 并保留 `data/`。

## 适用场景
- 个人创作者用同一界面反复精调提示词，批量生成风格一致的图像。
- 设计师利用颜色芯片和图库快速产出品牌素材，迭代参考图。
- 小团队在共享工作站上通过 API 模式统一管理多供应商 Key，并浏览历史结果。
- 需要本地数据留存的敏感项目，避免将原图上传至第三方网页。

## 同类对比
- 相比 ChatGPT 网页版：多任务并发、芯片/模板系统、本地历史、不受对话轮次限制。
- 相比 Stable Diffusion WebUI：无需下载本地模型、不占 GPU，但依赖远程 API 延迟；提示词工具更结构化。
- 相比 ComfyUI：学习成本更低，不需要连接节点，但牺牲了工作流的极致灵活性。
- 相比其他 OpenAI API 客户端：深度集成图像生成特化功能（图层编辑、格式自动转换、尺寸回退），而非通用聊天。
