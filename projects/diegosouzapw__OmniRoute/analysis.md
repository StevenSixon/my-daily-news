## 它是什么
OmniRoute 是一个开源免费的 AI 网关，通过单一端点（/v1）聚合 237+ AI 提供商（包括 90+ 免费层），为 Claude Code、Cursor、Cline 等 AI 编码代理提供无缝接入。内置 RTK + Caveman 压缩技术，可减少 15–95% 的令牌消耗（工具密集时平均约 89%），并附带智能故障转移、多策略路由、使用仪表盘、MCP/A2A 协议支持。

## 为什么火
1. 聚合超大量免费 AI 提供商（1.6B+ 免费令牌/月），按池去重计算，不虚标。
2. 堆叠压缩大幅降低编码代理的成本与限流风险。
3. 毫秒级自动故障转移，一台不可用立即切换，保障编码不中断。
4. 兼容超过 24 种编码 AI 工具，一键配置，无需单独接入各家 SDK。
5. 本地优先，可自托管，保护隐私。

## 技术栈
- 语言：TypeScript
- 分发：npm 包、Docker 镜像、Electron 桌面版、PWA
- 后端：Node.js（基于 Next.js 的仪表盘与 WebSocket 实时传输）
- 协议：OpenAI API 兼容，可桥接 Anthropic、Gemini、Responses 等格式
- 测试：21000+ 单元测试覆盖

## 核心能力
- **237 家提供商**，其中 90+ 提供免费层，11 家永久免费
- **RTK + Caveman 压缩**：堆叠压缩，节省 15–95% 令牌（平均约 89%），显著降低工具输出的令牌占用
- **4 级自动故障转移**：订阅 → API → 廉价 → 免费，配额耗尽或出错时自动切换
- **17 种路由策略**：成本优先、延迟、权重等组合
- **多模态 API**：支持文本、图像等多类输入
- **MCP/A2A**：95 个 MCP 工具及 A2A 互通
- **全功能仪表盘**：实时使用量、配额百分比、成本钻取、WebSocket 推送
- **代理与隐身**：3 级代理、TLS 指纹伪装，绕过地区封锁

## 适用场景
1. AI 编码工作流：接入 Claude Code、Codex、Cursor、Cline、Copilot 等，获得海量免费后备，防止因限流中断编码。
2. 开发者自托管：在本地或服务器上运行，统一管理多个 AI API 密钥和配额。
3. 团队内部网关：提供统一端点、用量监控和成本控制，避免个人订阅浪费。
4. 受限地区用户：通过代理和 TLS 伪装突破网络限制。

## 同类对比
与 OpenRouter、LiteLLM、Portkey 等 AI 网关相比，OmniRoute 的突出优势：
- 免费提供商聚合规模更大，并明确去重计算，可信度更高。
- 内置令牌压缩，非仅转发，实际降低每次请求的成本。
- 深度集成 AI 编码工具生态，对 Claude Code/Codex/Cursor 等有专用适配。
- 全图形仪表盘与配额钻取，直观管理免费额度。
- 同类产品多偏向付费提供商统一管理，OmniRoute 更侧重“免费+压缩+容错”的极客生存路线。
