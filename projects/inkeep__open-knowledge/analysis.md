## 它是什么
OpenKnowledge 是一个面向极客的 AI 原生 WYSIWYG Markdown 编辑器，专为构建第二大脑和 LLM Wiki 设计。它提供 macOS 桌面应用和 Web UI，完全本地化运行，数据控制权在用户手中；通过 MCP 和技能系统，可以直接与 Claude、Codex、Cursor 等 AI 桌面代理协作，让 AI 主动读写你的知识库。

## 为什么火
- **本地优先，隐私无忧**：文件都在你的磁盘上，无需云端同步即可 AI 协作，完全免费。
- **真实的所见即所得**：像 Notion 一样编辑，却基于 Markdown/MDX 纯文本，没有平台锁定。
- **深度 AI 集成**：不再是简单的聊天窗口，而是将知识库作为代理的“大脑”，允许 Claude/Codex 直接查询、编辑文档。
- **开发者友好**：支持嵌入 HTML 组件，适合编写技术规范与可视化报告，通过 Git 无缝同步和团队共享。
- **开箱即用**：自动检测已安装的 AI 代理工具，初始化 MCP 配置和技能，降低上手门槛。

## 技术栈
基于 TypeScript，使用 Next.js (React + Tailwind CSS) 构建 Web 编辑界面，并通过 Electron 封装为 macOS 桌面应用。CLI 工具 `ok` 负责项目初始化，自动配置 MCP 服务器和技能文件，利用 Git 进行版本控制和协作。内置终端 UI (TUI) 方便在终端中操作，整体架构强调文件系统为数据库、AI 代理为第一公民。

## 核心能力
- **WYSIWYG Markdown 编辑**：实时渲染，支持 MDX 组件。
- **AI 代理协作**：选中文本后通过“Ask AI”将上下文传递给终端中的 AI 代理，代理可基于文档内容回答问题或生成内容。
- **MCP 与技能系统**：为代理提供标准化的知识访问接口，让 AI 能够搜索、引用知识库，构建真正的第二大脑。
- **文件管理与图谱**：多标签、文件导航、全局搜索、Wiki 链接图谱。
- **一键共享**：基于 Git/GitHub 的无代码同步，团队可直接用 Git 工作流协作。
- **跨平台**：macOS 桌面应用，其他系统通过 Web 界面使用。

## 适用场景
- 个人第二大脑：笔记、文档、灵感收集，AI 辅助整理与检索。
- LLM 知识引擎：为代码代理提供项目文档、规范，让代理更了解你的代码库。
- 技术团队文档站：利用 MDX 组件和 Git 同步，构建私有或公共文档。
- 极客的日常编辑器：替代 Obsidian/Notion，享受本地文件的自由与 AI 原生体验。

## 同类对比
- **vs Obsidian**：OpenKnowledge 更侧重 AI 原生和所见即所得，无需大量插件即可与 Claude 等代理集成；Obsidian 在纯文本编辑和社区生态上更成熟。
- **vs Notion**：完全本地化，无云端锁定，开源，可自由同步与扩展。
- **vs 传统 Markdown 编辑器（Typora 等）**：增加了 AI 代理作为知识处理引擎，而不仅仅是静态文档编辑。
- **vs Foam/Roam**：缺少强大的双向链接，但提供了更丰富的 AI 交互和协作模式。
