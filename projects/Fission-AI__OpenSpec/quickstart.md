## 安装

**前置条件**：Node.js ≥ 20.19

```bash
npm install -g @fission-ai/openspec@latest
```

## 初始化

在项目目录中：

```bash
cd your-project
openspec init
```

## 开始使用

- 不确定要做什么：在 AI 助手对话中输入 `/opsx:explore`，描述你的想法，AI 会分析代码并给方案。
- 已有明确需求：输入 `/opsx:propose 添加暗色模式`，AI 会生成结构化的变更目录。
- 查看生成的任务并实施：输入 `/opsx:apply`。
- 完成后归档：输入 `/opsx:archive`。

## 升级

```bash
npm install -g @fission-ai/openspec@latest
cd your-project && openspec update
```
