## 它是什么
一个基于大语言模型的多市场股票智能分析系统，支持 A 股、港股、美股、日股、韩股，每日自动拉取行情与新闻数据，由 AI 生成包含评分、趋势、买卖点位、风险警报和催化因素的决策仪表盘，并推送到企业微信、飞书、Telegram 等渠道。提供 Web/桌面工作台，支持 Agent 策略问股与智能股票导入。

## 为什么火
- 零成本定时运行：利用 GitHub Actions 实现工作日自动分析，无需服务器。
- 多市场覆盖：原生支持五大市场，数据源丰富，适合跨市场投资者。
- 高度集成：内置多种 AI 模型（OpenAI 兼容、Gemini、Claude、本地 Ollama 等）和新闻搜索源，配置灵活。
- 社区活跃：Star 超 5.1 万，持续更新，文档齐全（中英双语），有多篇部署视频教程。

## 技术栈
- 语言：Python 3.10+
- 核心框架：FastAPI（Web 后端）、GitHub Actions（定时任务）、Docker（自托管部署）
- 数据源：TickFlow、AkShare、Tushare、YFinance、Longbridge 等
- 模型接入：Anspire、AIHubMix、Gemini、OpenAI、DeepSeek、Claude、本地模型等
- 通知渠道：企业微信、飞书、Telegram、Discord、Slack、邮件
- 前端：Web 工作台（浅/深色主题）、桌面端（Electron 打包）

## 核心能力
- 每日决策仪表盘：AI 生成核心结论、评分、趋势判断、买卖点位、风险警报、催化因素、操作清单。
- 多市场数据聚合：涵盖行情、K 线、技术指标、资金流、筹码、新闻、公告、基本面，日韩市场提供基础行情降级展示。
- Web/桌面工作台：手动分析、历史报告、回测、持仓管理、智能导入、配置管理。
- Agent 策略问股：多轮追问，内置 15 种策略（均线、缠论、波浪等），支持实时行情与新闻调用。
- 零成本自动化：通过 GitHub Actions 定时执行，支持非交易日跳过、断点续传。
- 多通知渠道推送：支持 Markdown 报告和图片模式。

## 适用场景
- 个人投资者每日快速获取自选股综合评估，辅助决策。
- 跨市场（A/港/美/日/韩）交易者的一站式分析看板。
- 想学习 LLM 在金融分析中应用的开发者，可基于开源代码二次开发。
- 量化新手尝试轻量级自动化分析，避免复杂回测系统。
