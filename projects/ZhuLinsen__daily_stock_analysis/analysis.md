## 它是什么

**daily_stock_analysis** 是一个基于 AI 大模型的个人股票决策助手，能每日自动拉取多市场行情、实时新闻、技术指标，由 LLM 生成包含评分、趋势、买卖点位、风险警报和操作清单的「决策仪表盘」，并通过企业微信、飞书、Telegram、Discord、邮件等渠道自动推送到你的设备上。

项目支持 A股、港股、美股、日股、韩股（日韩尚处 MVP 阶段），19 种数据源 + 14 种搜索/舆情接口，15 种内置策略的 Agent 问答，以及 Web 工作台、桌面客户端、Docker 一键部署。

## 为什么火

- **零成本自动化**：依赖 GitHub Actions 即可实现工作日内每日 18:00 自动运行，无需服务器，Fork + 配置 Secrets 5 分钟上线。
- **全市场覆盖**：A/港/美/ETF 及近期加入的日韩股（`.T`、`.KS`、`.KQ`），适配不同市场的数据边界（筹码、龙虎榜等不支持的板块自动降级为 `not_supported`）。
- **AI 决策闭环**：v3.23.0 起引入 `DecisionSignal` 机制，打通 AI 建议→报告展示→用户反馈→后验通知→组合风险监控的完整链路。
- **开箱即用的 Agent**：既可在 Web 界面追问均线、缠论、波浪等策略，也支持自定义策略文件及多 Agent 编排。
- **成熟的开发生态**：44000+ Star，活跃维护，配套 AlphaSift（选股）和 AlphaEvo（策略进化）系列项目，并已接入 HelloGitHub 等平台推荐。

## 技术栈

- **语言/框架**：Python 3.10+，FastAPI 提供 Web 与 API 服务，Docker 支持
- **AI 层**：支持 Anspire、AIHubMix、Gemini、Claude、OpenAI 兼容（DeepSeek、通义千问）、Ollama 本地等，通过统一适配器调度
- **数据源**：行情（AkShare、Tushare、Pytdx、Baostock、YFinance、Longbridge）、新闻搜索（Anspire、SerpAPI、Tavily、博查、Brave、MiniMax、SearXNG）、社交舆情（Reddit/X/Polymarket）
- **自动化与部署**：GitHub Actions、Docker、定时任务，Token 用量监控与调用可观测性埋点
- **前端展示**：Web 工作台（深色/浅色主题）、桌面端打包、实时任务进度、回测、持仓管理、智能导入与补全

## 核心能力

1. **每日决策报告**：核心结论、多空评分、趋势判断、买卖点位、风险警报、催化因素、操作检查清单
2. **多市场数据聚合**：跨 A/港/美/日/韩的行情、K线、技术指标、资金流、筹码、新闻、公告和基本面，日韩市场在资金流、龙虎榜、板块等高阶数据上自动降级
3. **Agent 策略问股**：支持多轮追问，内置均线、缠论、波浪、趋势、热点、事件、成长等 15 种策略，可自定义策略文件
4. **闭环信号追踪**：v3.23.0 新增的 DecisionSignal 将 AI 买卖建议提取为结构化信号，串联报告展示、用户反馈、后验验证和组合风险监控
5. **多通道推送**：企业微信、飞书、Telegram、Discord、Slack、邮件，支持分组推送与 Markdown 转图片
6. **零成本定时运行**：GitHub Actions 自动执行，非交易日跳过，支持断点续传
7. **Web 与桌面工作台**：手动分析、历史报告查看、完整 Markdown、回测、持仓、智能导入，一键启动 `python main.py --webui`

## 适用场景

- 个人投资者日常盯盘，希望减少信息筛选时间，获得结构化的多因子决策参考
- 小型投研团队需要低成本的自动化晨会报告与推送
- 量化爱好者研究 AI 驱动的策略生成、Agent 对话与信号反馈闭环
- 学习或教学中使用开源系统演示多市场金融数据聚合与 LLM 应用

## 同类对比

与一般开源量化框架（如 vnpy、Backtrader）相比，daily_stock_analysis 更侧重“AI 解读 + 日常推送”而非交易执行；与 StockBot 等聊天机械人相比，它直接提供全市场数据聚合、决策仪表盘和 Agent 策略组合，并支持零成本的 GitHub Actions 部署。与作者的另一项目 AlphaSift（选股）和 AlphaEvo（策略进化）形成“选股→每日分析→策略验证”的工具体系。

## 版本动态

近期 v3.23.0 版本关键更新：
- DecisionSignal 闭环：AI 建议可被报告提取、Web 展示、用户反馈、后验计算和告警推送，进入可追踪的组合风险管理流程。
- 新增合规新闻源（RSS/Atom 与 NewsNow），分析、Agent 和大盘复盘可在资讯源故障时复用本地缓存证据。
- 日韩股票 MVP 支持：通过 YFinance 获取 `.T`、`.KS`、`.KQ` 标的日线行情与技术指标。
- 可观测性增强：Token 用量监控看板、legacy LLM 调用遥测、消息稳定性审计。

## ℹ️ 置信度与信息盲区

- 置信度：**high**
- 信息盲区：日韩股票分析目前标注为 MVP，资金流、龙虎榜、板块等高阶数据因市场边界降级为 `not_supported`，实际可用深度尚未公布基准对比；LLM 生成决策的准确率、回测表现或用户反馈统计未提供；Agent 策略有效性及自定义策略的实际表现无量化说明；推送到多个渠道时是否存在限流或延迟情况不明；大数据量（例如数百只自选股）下的分析耗时和成本没有给出参考值
