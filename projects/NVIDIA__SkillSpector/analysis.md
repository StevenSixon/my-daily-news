## 它是什么
SkillSpector 是 NVIDIA 开源的一款 AI agent 技能安全扫描器，面向 Claude Code、Codex CLI、Gemini CLI 等平台下发的技能文件（如 SKILL.md）。它通过静态模式匹配与可选的 LLM 语义分析，在安装前检测潜在漏洞、恶意行为和安全风险，并输出终端报告、JSON、Markdown 或 SARIF 格式的结果，帮助开发者回答“这个技能安全吗？”

## 为什么火
研究显示 26.1% 的 AI 技能包含漏洞，5.2% 表现出可能的恶意意图。随着 AI agent 快速普及，技能作为第三方扩展被赋予隐式信任，缺乏前置审查机制。SkillSpector 恰好填补了这一真空：开箱即用、无需复杂配置，且直接对接主流 LLM 提供商，一经发布即获得广泛关注（Star 数超 8k），成为 agent 安全领域的先行工具。

## 技术栈
- 语言：Python 3.12+
- 核心分析引擎：AST 分析、YARA 签名、正则模式、污点追踪（taint tracking）
- 静态规则库：64 种漏洞模式，覆盖 16 个类别（注入、提权、供应链、过度代理等）
- LLM 集成：通过 OpenAI 兼容接口调用 GPT、Claude、NVIDIA 模型或本地 Ollama/vLLM 等进行语义深度分析
- 漏洞情报：SC4 模块实时查询 OSV.dev 获取已知 CVE 信息
- 构建与运行：支持 pip/uv 安装，提供 Dockerfile 免 Python 环境运行
- 输出格式：Terminal、JSON、Markdown、SARIF，方便 CI/CD 集成

## 核心能力
1. **多输入溯源**：扫描 Git 仓库、URL、zip 包、本地目录或单文件
2. **64 个漏洞模式**：覆盖提示注入、数据窃取、权限提升、供应链、过度代理、输出处理、系统提示泄露、内存投毒、工具滥用、恶意代理、触发器滥用、危险代码（AST）、污点追踪、YARA 规则、MCP 最小权限/MCP 工具投毒等 16 大类
3. **双阶段分析**：静态快速扫描 + 可选的 LLM 语义评估，命中率更高
4. **实时 CVE 查询**：对依赖库实时匹配 OSV.dev 漏洞库，离线时自动降级
5. **风险评分与建议**：0‑100 分风险评分，附带严重程度标签和处置建议
6. **多格式报告**：SARIF 输出适合 CI/CD 管道与 IDE 集成

## 适用场景
- 开发者在安装第三方 AI agent 技能前的安全检查
- 技能市场/注册中心的上架审核自动化
- CI/CD 流程中自动扫描 agent 技能更新的依赖与行为风险
- 企业安全团队对内部定制的 AI 工作流技能进行审计

## 同类对比
目前市场上专门针对 AI agent 技能的安全扫描工具尚属空白。传统 SAST/DAST 工具（如 Snyk、CodeQL）侧重传统应用代码，未对 agent 特有的隐式指令、上下文投毒、工具链滥用等风险建模。SkillSpector 通过细化的 agent 威胁分类和 LLM 辅助语义理解，建立了新品类标杆。与同领域早期 idea 项目（如 agent-security-linter）相比，SkillSpector 覆盖类别更全、支持 LLM 实时分析且输出格式更企业化。
