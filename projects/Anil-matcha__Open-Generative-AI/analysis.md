## 它是什么
Open Generative AI 是一个 MIT 许可的开源 AI 图像与视频生成工作室，自称“无限制的 AI 视频平台替代品”。它集成了 200+ 主流模型（Flux、Midjourney、Kling、Sora、Veo 等），覆盖文生图、图生图、文生视频、图生视频、唇形同步等场景，并提供桌面应用（Windows/macOS/Linux）与在线版本。

## 为什么火
- **零内容过滤**：无提示词拒绝、无安全护栏，提供完全自由的生成体验，这是其他商业平台不具备的。
- **真正的自托管**：基于 Electron 的桌面版支持一键安装，且自带 sd.cpp 和 Wan2GP 两种本地推理引擎，可选择完全离线运行部分模型，无需 API 密钥。
- **模型丰富度碾压**：聚合 200+ 模型，覆盖文生图、视频、音频、唇形同步等任务，集成度远超单模型工具。
- **活跃更新与社区**：v2.0.0 新增音频工作室、设计助手、本地模型存储自定义等，社区 Discord 活跃。

## 技术栈
- **前端/桌面壳**：JavaScript（推测 React + Vite），打包为 Electron 应用，支持 Windows、macOS、Linux。
- **本地推理引擎**：sd.cpp（基于 stable-diffusion.cpp，Metal/CUDA/Vulkan 等后端）用于图像模型；Wan2GP（需要自建 Python+PyTorch 服务端）用于视频和大图模型。
- **远程服务**：主要功能通过 MuAPI 调用云端 API，需要提供 API Key。

## 核心能力
- **图像/视频生成**：文生图、图生图、文生视频、图生视频，支持最多 14 张参考图输入。
- **唇形同步**：9 个模型，可驱动肖像说话。
- **新工作室**：音频生成、Vibe Motion 动效、视频剪辑、Design Agent 助手等。
- **本地模型**：桌面版可下载选定的图片模型（如 Z-Image Turbo、Dreamshaper 8 等），完全在本地推理。
- **国际化**：中英文切换。

## 适用场景
- 需要绝对无限制、无审查的内容创作（艺术、教育、研究等）。
- 希望摆脱月费订阅、掌控生成管线的独立创作者。
- 愿意折腾本地模型、追求隐私或离线可用性的极客。
- 快速原型制作：通过浏览器直接试用所有最新模型。

## 同类对比
| 特点 | Open Generative AI | Runway / Pika | Midjourney |
|------|-------------------|---------------|------------|
| 内容过滤 | 无 | 严格 | 有 |
| 模型数量 | 200+ | 有限 | 单一 |
| 本地推理 | sd.cpp / Wan2GP | 无 | 无 |
| 自托管 | 完全支持 | 不支持 | 不支持 |
| 开源 | MIT | 闭源 | 闭源 |
| 费用 | 免费（需 MuAPI 密钥或本地运行） | 付费订阅 | 付费 |

相较于商业产品，它的自由度、集成度和可定制性极高，但部分功能依赖第三方 API，本地模型对硬件有要求。
