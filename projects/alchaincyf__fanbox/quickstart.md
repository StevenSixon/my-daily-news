## 安装
1. 从 [GitHub Releases](https://github.com/alchaincyf/fanbox/releases/latest) 下载最新 `.dmg`（Apple Silicon）。
2. 拖入 Applications 文件夹，首次打开若提示未验证开发者：右键 → 打开 → 确认。

## 最小可用
- 打开 FanBox，在左侧文件区通过 `⌘K` 搜索或直接浏览打开项目文件夹。
- 内嵌终端中运行 `claude` 或 `codex` 即可开始 vibe coding。
- 文件改动会实时高亮卡片，右侧/下侧终端可拖入文件路径作为上下文。

## 依赖前提
- macOS（Apple Silicon arm64）
- 需自行安装 Claude Code 或 Codex CLI（非 FanBox 内置）
