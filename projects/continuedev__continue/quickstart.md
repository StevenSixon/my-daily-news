## 安装
由于仓库已只读，仍可安装最终版：
- VS Code：市场搜索 “Continue” 或访问 [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=Continue.continue)
- CLI：`npm install -g @continuedev/cli`
- JetBrains：从 [GitHub Releases](https://github.com/continuedev/continue/releases) 下载插件（官方推荐改用 CLI）

## 最小可用示例
安装后，在编辑器或终端中启动 Continue，按文档配置即可使用。具体配置需参考官方文档（README 仅指向 docs.continue.dev）。

## 依赖前提
- Node.js 环境（CLI）
- VS Code 或 JetBrains IDE（插件）
- 可能需要接入自部署或第三方 LLM，但细节未在 README 中提供
