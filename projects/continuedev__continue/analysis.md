## 它是什么
Continue 是一个开源的编码代理，提供 CLI、VS Code 扩展和 JetBrains 插件。它曾作为“先驱性”的 AI 开发工具被开发，让开发者在本地或自选环境中使用 AI 辅助编码。

## 为什么火
在 GitHub 上获得了 34k+ Star，反映出社区对开源、可控的 AI 编码助手的强烈需求。它曾与闭源方案（如 GitHub Copilot）形成对比，吸引重视数据隐私和定制性的开发者。

## 技术栈
- 语言：TypeScript
- 分发：VS Code 扩展、npm 包（CLI）、JetBrains 插件
- 许可证：Apache 2.0

README 未详述功能。文档站点可能覆盖配置自定义模型、聊天、代码补全、代理任务等，但本分析仅基于提供的 README 截断内容，无法确认具体能力。从发布的“2.0.0 final release”看，曾提供了完整的编码代理体验，但细节缺失。

## 适用场景
- 需要离线或私有化部署 AI 编程助手的团队
- 研究和学习 AI 编码代理架构的开发者
- 对闭源 Copilot 不信任、想自托管增强控制的用户

## 同类对比
- vs GitHub Copilot：Copilot 为闭源云服务；Continue 开源、可自托管，但现已停更，生态和体验可能落后。
- vs Tabby / Codeium 等：同样自托管，但 Continue 更早获得关注，现已进入只读状态。
