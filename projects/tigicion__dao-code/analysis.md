## 它是什么

Dao Code 是一个开源的终端编码代理，专为 DeepSeek V4 设计，利用字节稳定前缀和缓存复用实现跨会话记忆与自修正，几乎不增加 token 成本。受 Claude Code 启发，MIT 开源，并针对中国开发者优化（无需代理即可访问）。

## 为什么火

以极致成本著称：7 个真实 OSS 修复任务总花费仅 ¥1.07，缓存命中率 95.8%，比 Claude Opus 便宜 30 倍。功能齐全（Memory、Reflection、MCP、Skills），且全开源无访问限制。

## 技术栈

- 语言：TypeScript
- 运行时：Node.js ≥20
- 终端 UI：Ink (React)
- 模型：DeepSeek V4 Pro/Flash
- 扩展：MCP、Hooks、自定义子代理

## 核心能力

- **上下文缓存工程**：字节稳定前缀，Fork 复用主缓存，自动压缩，溢出落盘
- **跨会话记忆**：启动时自验证，陈旧淘汰，按需读取
- **反思层**：挑战者/聚焦者/回复挑战者，共享缓存
- **长任务鲁棒**：影子 git 检查点，任务清单，子代理通信
- **终端体验**：流式思考，权限门控，`@` 文件引用，ESC 中断
- **扩展系统**：兼容 Claude Code 配置，自动适配第三方技能

## 适用场景

- 中国大陆开发者终端编码、调试、重构（无需梯子）
- 长期自主任务（如 `--goal` 模式自动修复 CI 失败）
- 成本敏感团队，需要可审计的 token 消费

## 同类对比

- 比 Claude Code：成本低 30 倍，开源，中国直连
- 比 Aider/Cursor：终端原生，缓存优化的记忆系统更适合大上下文
