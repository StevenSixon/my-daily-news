## 它是什么
Dao Code (`dao`) 是一个终端原生 AI 编码助手，专为 DeepSeek V4 打造：读取、编辑代码、执行命令、修复 bug，所有操作在终端内完成，包含推理流式输出、工具调用审核闸门、跨会话记忆等完整功能。开源 (MIT)、中国首发、注册即用，无需科学上网。

## 为什么火
- **成本杀手**：利用 DeepSeek 前缀缓存（命中价格 ≈ 未命中的 1/120），精心设计字节稳定的系统前缀、记忆和自省层，使缓存命中率达 95.8%；在 7 个真实开源 bug 修复测试中，每次功能开发总成本 ¥0.07-0.21，平均 ¥0.15，与 Claude Opus 4.8 比便宜约 30 倍。
- **反幻觉记忆**：跨会话记忆在每次启动时自动用最新代码验证，过期事实被剔除，变化被标记，不盲目堆积历史。
- **可及性**：无账号门槛（DeepSeek 直接注册）、无配额抢夺、国内直连。

## 技术栈
TypeScript + Node.js (≥20)，终端 UI 使用 Rich/Ink 渲染，核心是流式对话循环与工具调用引擎。深度依赖 DeepSeek API 的缓存机制，自行管理前缀布局和 fork 复用。支持 Skills/MCP/Hooks/插件等扩展生态。

## 核心能力
- **前缀缓存工程**：系统提示词、工具表、内存等保持字节不变，自省与记忆更新在 fork 上完成以复用主缓存。
- **自修正层**：Challenger（对失败倾向质疑前提）、Refocuser（长任务防跑偏）、Reply-challenger（用户重复问题时触发）。
- **长任务稳健性**：自动压缩（接近上下文限制时）、影子 git 检查点（不污染 `.git`）、存活 TODO 列表、完成定义验证、卡顿检测与熔断。
- **安全底线**：宪法式优先级（安全/真值 > 用户指令 > 代理自身策略），即使第三方 skill 也无法绕过。
- **扩展系统**：24 个内置工具、分层权限（allow/ask/deny/auto）、MCP 双传输支持、生命周期钩子、子代理、自定义斜杠命令。
- **终端体验**：Taiji 启动动画、暗/亮主题适配、`@` 文件引用 + Tab 补全、ESC 中断、非 TTY 纯文本回退。

## 适用场景
- 国内开发者：无需 VPN、直接可用，成本极低。
- 个人/小团队：想在终端快速 Fix bug、阅读代码、运行自动化重构。
- 极客/研究者：研究缓存优化、代理自我修正、长任务控制策略的代码级参考。
- CI/CD 集成：`dao "任务"` 单次执行模式可直接嵌入脚本。

## 同类对比
- **Claude Code**：功能镜像（工具/权限、扩展体系），但需 Anthropic 账号和网络，成本高 18-30 倍。Dao Code 可通过配置兼容 Claude Code 配置。
- **GLM Coding Plan**：配额稀缺，需抢购。
- **Aider / OpenHands**：未直接对比，但 Dao Code 的核心差异化在于极致的缓存成本控制和自我验证记忆，而非单纯的内联编辑能力。
- **Cline / Continue**：前者偏向 IDE 插件，Dao Code 是纯终端代理，适合 SSH/服务器环境。
