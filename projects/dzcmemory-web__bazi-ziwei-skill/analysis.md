## 它是什么
一个遵循 SKILL.md 开放标准的命理分析 Agent 技能。内置八字四柱、紫微十二宫、大运流年排盘算法，结合 LLM 进行三种模式（八字独立/紫微独立/八字+紫微综合印证）的分析，并可一键生成水墨风 HTML 命盘海报。

## 为什么火
传统用 LLM 直接排盘经常算错日柱、格局，导致分析失真。该项目将排盘交给确定性算法，LLM 只负责分析，从根本上保证了基础数据的准确。同时，它引入八字与紫微两套独立体系的交叉印证，提升了分析的深度与可信度，对玄学爱好者和 AI Agent 开发者都有吸引力。

## 技术栈
- 语言：TypeScript
- 核心算法：基于 Yiqi 八字紫微排盘系统（MIT），农历转换依赖 lunar-typescript
- Agent 兼容：Claude Code、Codex、Cursor、Hermes、OpenClaw 等
- 输出形式：Markdown 长文 + 单文件 HTML 海报（模板注入）

## 核心能力
- 精准排盘：不依赖 LLM 推算，由内置算法完成四柱、紫微宫位、大运流年
- 格局补层：在排盘基础上补充格局、旺衰、调候、刑冲合害等算法层信息
- 综合印证：对八字与紫微的结论进行交叉对账，识别人生窗口、判断冲突
- 双形态输出：深度分析长文（Markdown）或水墨风 HTML 命盘海报
- 隐私优先：全部本地计算，无需联网

## 适用场景
- AI 命理分析助手：集成到各类 Agent 中，为用户提供八字/紫微排盘与分析
- 民俗文化研究：作为可编程的排盘工具，批量验证命理理论
- 内容生成：自动生成带命盘的可视化海报，用于分享或展示

## 同类对比
- 纯 LLM 排盘：经常出错，缺乏确定性；本项目优先保证排盘准确
- 传统排盘网站/软件：单机使用，无法与 AI 分析流程结合；本项目以 Agent Skill 形式存在，可嵌入对话流
- 其他排盘开源库：多止步于排盘，不做综合印证与海报生成；本项目提供端到端体验
