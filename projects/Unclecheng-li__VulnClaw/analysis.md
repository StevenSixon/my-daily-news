## 它是什么
VulnClaw 是一个基于 LLM Agent + MCP（模型上下文协议）的 AI 渗透测试 CLI 工具。用户以自然语言描述目标，它能自动执行信息收集、漏洞发现、漏洞利用和报告生成全流程，适合授权渗透测试、CTF 竞赛和安全教学。

## 为什么火
项目 Star 数快速破千，原因在于切中了安全自动化与 LLM 结合的痛点：传统手工渗透耗时长，完全交给 AI 又容易出现幻觉和死循环。VulnClaw 用状态空间搜索和反幻觉闸门解决了 LLM 自主渗透的可靠性问题，并且支持 13 个国内大模型 Provider，降低了国内用户的使用门槛。

## 技术栈
- 核心语言：Python 3.10+
- LLM 集成：OpenAI 兼容协议，支持 DeepSeek、MiniMax、智谱、千问等国内模型
- 工具链：MCP 协议，内置 fetch/memory 等 MCP 服务，可对接 Chrome DevTools、Burp Suite
- 推理引擎：目标驱动求解（黑板图+OODA 循环）、结构化推理、自适应反思
- 界面：CLI/REPL/TUI/Web UI

## 核心能力
- **目标驱动求解引擎**：不使用固定轮数，以目标达成或探索前沿耗尽为终止条件
- **证据级反幻觉闸门**：所有结论必须在真实工具输出中验证，杜绝编造 flag 的假胜利
- **结构化推理 + 自适应反思**：已知事实和攻击链结构化沉淀，失败时渐进升级 payload 策略
- **漏洞检测插件体系**：低耦合插件运行时，结果合并入报告
- **MCP 工具链**：可扩展工具调用，协调 LLM 与外部能力
- **编解码/加解密**：内置 29 种编码/加密操作，LLM 精确调用
- **Python 代码执行**：内置沙箱（实验性），用于构造 payload 和解析响应
- **多模型切换**：一键切换 13 个 Provider，支持无 API Key 的 ChatGPT 订阅登录
- **TUI/Web UI**：提供图形化工作台和浏览器界面

## 适用场景
- **授权渗透测试**：快速摸底 Web 应用的常见漏洞
- **CTF 竞赛**：辅助信息收集、弱口令爆破、漏洞 PoC 编写
- **安全教学/红队演练**：演示自动化攻击链
- **持续性监测**：长时间深度扫描，周期自动生成报告

## 同类对比
- **GPT + Burp Suite 插件**：VulnClaw 是独立 Agent，不绑定特定抓包工具，且支持更多国内模型
- **PentestGPT**：同样用 LLM 驱动，但 VulnClaw 强调状态搜索和反幻觉，结构上更抗死循环
- **Burp BApps 自动化**：VulnClaw 可以对接 Burp MCP 服务，但自身是完整工作流
- **自建 LangChain Agent**：VulnClaw 开箱即用，内置渗透 Skill 和报告生成，无需自己搭链路
