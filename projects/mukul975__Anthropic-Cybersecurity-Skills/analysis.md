## 它是什么

Anthropic Cybersecurity Skills 是一个面向AI代理的开源网络安全技能库，包含 **754个结构化安全技能**，覆盖**26个安全领域**。每个技能遵循 [agentskills.io](https://agentskills.io) 开放标准，使用 YAML 前置元数据 + 结构化 Markdown 描述实际安全专家的操作流程。核心区别在于：它**不是脚本或检查清单**，而是专为AI代理设计的可执行知识。

该库将每一个技能同时映射到**6个安全框架**：MITRE ATT&CK、NIST CSF 2.0、MITRE ATLAS、MITRE D3FEND、NIST AI RMF 以及 MITRE F3（欺诈框架），实现跨框架统一覆盖，这是目前唯一一个做到这点的开源项目。

## 为什么火

全球网络安全岗位缺口达480万（ISC2 2024），AI代理被认为能弥补能力缺口，但通用LLM缺乏领域专用操作流程。该仓库直接提供**一线分析师的决策工作流**：何时、如何执行特定技术，前置条件、步骤与验证方法，而不仅仅是工具或漏洞利用代码。自发布以来迅速获得 **17.7k Star**，被广泛用于 Claude Code、GitHub Copilot 等主流AI编码助手及安全代理平台，因其**开箱即用**和**多框架合规映射**能力受到安全团队青睐。

## 技术栈

- **技能格式**：agentskills.io 标准，每个技能为独立的 Markdown 文件，包含 YAML frontmatter 声明元数据（如 `mitre_attack`, `nist_csf`, `atlas_techniques` 等）。
- **仓库结构**：主要为文档型仓库；标注语言为 Python（可能包含少量验证脚本），但核心价值在于结构化内容，不依赖特定运行时。
- **集成方式**：通过 `npx skills add` 或 `git clone` 即可为支持的 AI 代理平台提供知识注入。
- **验证**：利用 `mitreattack-python` 等库确保框架ID的正确性（如 ATT&CK v19.1），所有映射均经过官方数据源校验。

## 核心能力

- **26个安全领域全覆盖**：云安全（60）、威胁狩猎（55）、恶意软件分析（39）、数字取证（37）、渗透测试（23）等，每个领域提供数十个实战技能。
- **六框架深度映射**：单个技能可同时关联攻击行为（ATT&CK）、防御对策（D3FEND）、AI对抗威胁（ATLAS）、风险管理（NIST AI RMF）等，满足合规与防御双重需求。
- **AI原生工作流**：技能结构包含前置条件、执行步骤、验证方法，可直接被AI代理解析并执行，而非简单的工具描述。
- **跨平台兼容**：支持 26+ 个AI代理平台，包括 Claude Code、GitHub Copilot、Cursor、Windsurf、OpenAI Codex CLI、Gemini CLI 等。

## 适用场景

- **安全运营中心（SOC）**：借助AI代理快速进行告警研判、日志分析、事件响应，提升效率。
- **红队/渗透测试**：构建自动化攻击模拟工作流，并参照 ATT&CK 进行战术映射。
- **云安全治理**：针对 AWS/Azure/GCP 的加固与合规检查，自动生成修复建议。
- **AI安全开发**：为自研安全Agent或Copilot提供专业知识库，减少从零构建的成本。
- **学术研究**：用于研究AI代理在网络安全任务中的表现，或作为教育训练材料的来源。

|------|------|------------|
| Atomic Red Team | ATT&CK 测试用例 | 提供可执行测试脚本，但非AI代理可读的工作流级技能，且无多框架映射。 |
| Sigma 规则库 | SIEM检测规则 | 专注于检测逻辑，缺少操作流程、框架映射，不面向AI代理。 |
| Awesome Security Lists | 资源索引 | 链接集合，无结构化技能定义，无法直接注入AI代理。 |
| 本库 | AI代理知识库 | **唯一**将安全技能结构化并跨六大框架映射的AI原生知识库。 |

## 版本动态

- **v1.2.0（2026-04-06）**：新增 MITRE ATLAS v5.5、MITRE D3FEND v1.3、NIST AI RMF 1.0 的映射，实现五项框架覆盖。每个 SKILL.md 添加了专门的框架字段。
- **MITRE F3 集成**：随后引入 MITRE 反欺诈框架 (F3) v1.1，增加 94 个欺诈相关技能映射，覆盖“定位”和“获利”两大新战术。
- **持续更新**：项目保持与 ATT&CK v19.1 等框架版本同步，skill 数量维持在 754 个，但映射随框架演进不断修订。
