## 它是什么

AI Berkshire 是一套基于 Anthropic Claude Code 的投资研究 Skill 集合，将巴菲特、芒格、段永平、李录四位价值投资大师的方法论系统化为 16 个可调用的命令（/investment-research 等）。每个命令内部启动多 Agent 并行研究，各自独立搜索、交叉验证，最终由 Team Lead 综合输出带有明确结论和价格区间的投研报告。

核心思想：一个人 + Claude Code = 一个专业投研团队。提供了实盘两年大幅跑赢标普500和恒生指数的收益数据（2024年+69.29%，2025年至今+66.38%），并附富途账户截图增强可信度。

## 为什么火

> Star 1757，2026年4月发布后迅速获得关注。

- **解决真实痛点**：直接问 AI “某股票值不值得买”会得到模棱两可的分析，AI Berkshire 强制输出通过/不通过/灰色地带，含具体价格区间和分层投资建议，杜绝“太极回答”。
- **四大师视角对抗**：段永平看商业模式、巴菲特看财务与护城河、芒格逆向思考风险、李录看长期确定性，四个视角会产生真实的矛盾与张力，避免单一视角盲点。
- **可复现的研究流程**：同样的输入保证结构和深度一致的输出，支持公司横向对比和时序跟踪。
- **内置反偏见机制**：信息丰富度评级、芒格式逆向检验、快速否决清单、反共识检查、留白原则等多层防骗设计，让 AI 不自作聪明的幻觉。
- **多 Agent 并行**：/investment-team 启动4个独立 Agent 同时研究一家公司，每个 Agent 各自进行完整搜索和独立结论，相当于4倍搜索量和信息源。

## 技术栈

- **载体**：Claude Code（Anthropic 的命令行 AI 编程/研究工具，需 npm 全局安装）。
- **Skill 实现**：一系列 Markdown 格式的提示文件（skills/*.md），用户复制到 ~/.claude/commands/ 后即可通过 / 命令调用。
- **工具支撑**：Python 脚本用于精确金融计算（如 decimal.Decimal 计算市值、DCF），部分工具在 tools/ 目录下，确保关键数据多源交叉验证、不依赖 LLM 心算。
- **架构**：三层设计——Skill 层（16个研究入口）、Agent 层（每个 skill 内4个 Agent 并行）、工具层（精确计算和实时检索）。
- **语言标签**：Python（因辅助工具）；实际主体为 Claude Code 的 Markdown 指令集。

## 核心能力

1. **深度研究类**（5个）：/investment-research（四大师综合）、/investment-team（4 Agent 并行）、/management-deep-dive（管理层深挖）、/private-company-research（未上市侦探式研究）、/deep-company-series（8篇长文系列拆解）。
2. **财报分析类**（2个）：/earnings-review（只读原始财报）、/earnings-team（四大师并行解读财报并输出可发布文章）。
3. **行业筛选类**（4个）：/industry-research（产业链全景扫描）、/industry-funnel（全市场漏斗筛选至3家）、/quality-screen（7条硬指标去劣删除）、/investment-checklist（巴菲特的六关快速筛选，10分钟决定是否深入）。
4. **持仓管理类**（3个）：/portfolio-review（组合仓位优化）、/thesis-tracker（投资论文持续验证）、/news-pulse（股价异动10分钟归因）。
5. **思维工具类**（2个）：/dyp-ask（段永平问答模式）、/financial-data（数据获取与交叉验证规范）。

每个核心能力都具备：
- 强制输出“镜子测试”（五句话说清楚买入理由，说不完就不买）
- 三情景估值（乐观/中性/悲观）+ 反向DCF
- 多源数据交叉验证，市值手算校验
- 明确标注信息丰富度等级（A/B/C）和置信度

## 适用场景

- **个人投资者**想做严肃的个股/行业研究，希望借助 AI 克服认知偏差，获得可执行的决策依据。
- **价值投资学习者**想学习如何将四大师方法论落地为具体的研究流程。
- **分析师/研究员**寻找可复现的 AI 辅助投研框架，快速生成初稿或进行多视角验证。
- **需要快速筛选**：在 10 分钟内判断一家公司是否值得深入研究（/investment-checklist）。
- **组合管理**：定期审查持仓，跟踪投资论文是否被证伪。

## 同类对比

- **直接对话式 AI（ChatGPT/Claude 对话）**：缺少强制决策纪律，回答模棱两可；AI Berkshire 给出了结构化评分、镜子测试、具体买卖价格区间。
- **传统金融终端（Bloomberg/FactSet）**：数据全面但缺乏大师方法论的 AI 执行和多视角对抗，且成本极高；AI Berkshire 成本极低（只需 Claude Code 订阅）。
- **其他 AI 投研工具（如 FinChat、Koyfin 的 AI 功能）**：强调数据可视化和快速归因，但缺少以四大师为骨架的完整研究框架，也缺乏未上市公司研究和财报深度解读的强制流程。
- **自建 LangChain 金融 Agent**：灵活但需要大量开发工作，AI Berkshire 开箱即用，且有实盘业绩背书。
