## 它是什么
AI Berkshire 是一套搭配 Claude Code 或 Codex 的投资研究 Skill 集合，将巴菲特、芒格、段永平、李录四位价值投资大师的方法论系统化为可执行的 AI 研究流程。用户在终端里用斜杠命令（如 `/investment-research 腾讯`）即可获得多维度、可对比、带有明确结论和价格区间的投研报告。其背后是18个精心设计的 Skill（深度研究、财报精读、行业漏斗筛选、持仓管理、思维工具等），每个 Skill 内部都用多Agent并行方式模拟真实投研团队的协作与对立。

## 为什么火
项目在 GitHub 获得 4800+ Star，核心理由是它直面“直接问AI做不了投资决策”的痛点：普通 AI 输出中立面面俱到的分析，而 AI Berkshire 强制给出通过/不通过/灰色地带结论，并附上具体的价格区间和分层策略。真实账户在 2024/2025 年均大幅跑赢标普500和恒生指数，证明了框架的实战有效性。此外，它通过四大师视角对抗、芒格式逆向检验、“镜子测试”等机制构建了结构化的反偏见系统，这是同类个人投资辅助工具中少见的。

## 技术栈
- 核心语言：Python（精确金融计算使用 decimal.Decimal 而非 float）
- 平台：Claude Code（npm包）和 Codex（OpenAI）双兼容
- 安装方式：shell 脚本将 skill markdown 文件拷贝到对应目录
- 计算工具：Python 脚本用于市值校验、DCF 模型等
- 无独立后端，所有技能即 markdown 命令文件

## 核心能力
1. **多Agent并行投研**：`/investment-team` 同时启动4个Agent独立搜索、交叉验证、独立评分，Team Lead 综合。
2. **四大师框架**：段永平（商业模式）、巴菲特（财务与护城河）、芒格（逆向思考）、李录（长期文明确定性）四个维度评分。
3. **结构化反偏见**：信息丰富度评级、快速否决清单（8条红线）、反共识检查、留白原则。
4. **精确计算**：关键数据多源验证，市值手算核验，误差>1%告警。
5. **批量对比**：同一 Checklist 标准横向对比7家公司，评分一致可复现。

## 适用场景
- 个人投资者对 A/港/美股上市公司进行快速初筛和深度研究
- 行业全景扫描与产业链瓶颈识别
- 持仓组合管理与定期复盘
- 财报季快速精读并生成可发布文章
- 未上市公司侦探式信息搜集与分析

## 同类对比
相比直接问 Claude 或 ChatGPT，AI Berkshire 的优势在于流程纪律、强制结论和可复现性。与传统的 Bloomberg 终端或 Wind 相比，它不依赖昂贵数据源，更侧重定性分析和决策辅助而非数据展示。与 FinGPT 等开源金融 LLM 项目相比，它更强调投资哲学的落实和 Agent 协作流程而非模型微调。缺点是数据获取依赖 AI 搜索，准确性受限于公开信息。
