## 它是什么
AI Berkshire 是一套基于 Anthropic Claude Code 的价值投资研究 Skill 集合，将巴菲特、芒格、段永平、李录四位投资大师的方法论结构化为 16 个可调用命令。通过 `/investment-research 腾讯` 等指令，Claude 能按预定流程进行多维度分析，强制输出结论与价格区间，并支持 4 个 AI Agent 并行研究同一标的，模拟真实投研团队的对抗视角。

## 为什么火
项目在 GitHub 获得 3013 颗星。背后有连续两年实盘超额收益（2024 年 +69.29%、2025 年 +66.38%），大幅跑赢标普500 和恒生指数。它解决了普通投资者直接向 AI 提问时得到的“平衡但无用”分析的问题，用结构化流程、强制结论、多大师视角对抗和反偏见机制，让 AI 输出可做投资决策的深度报告。这种“一人成团”的投研范式吸引了大量价值投资者和 AI 应用探索者。

## 技术栈
- **语言/工具**：Python（用于财务精确计算，如 `tools/financial_rigor.py`），Skill 定义文件为 Markdown
- **核心依赖**：Anthropic Claude Code（需通过 npm 安装），利用 Claude 的指令跟随与 Agent 能力
- **并行机制**：通过 Claude Code 的命令系统触发多个独立 Agent 会话，各自搜索并交叉验证
- **架构**：三层设计——Skill 层（16 个入口）→ Agent 层（4 个并行分析师）→ 工具层（精确计算、实时检索、报告抽检）
- **安装方式**：纯命令行，将 Skill 的 `.md` 文件复制到 Claude Code 的 commands 目录即可

## 核心能力
1. **16 个结构化 Skill**：覆盖深度研究（个股、未上市公司）、财报分析、行业筛选、持仓管理、思维工具
2. **强制结论输出**：每个分析必须给出“通过/不通过/灰色地带”，并附具体价格区间，禁止模棱两可
3. **四大师视角对抗**：段永平看生意、巴菲特看财务、芒格逆向、李录看长线，产生真实的碰撞
4. **反偏见机制**：信息丰富度评级、芒格式逆向检验、快速否决清单、反共识检查、留白原则
5. **财务数据精确性**：市值手算校验、Python Decimal 代替 float、多源交叉验证（偏差 >1% 报警）
6. **可复现流程**：同一输入产生结构一致、评分可比的结果，支持公司横向对比和定期重检
7. **多 Agent 并行**：`/investment-team` 启动 4 个独立 Agent 同时研究，相当于四倍信息源和独立视角
8. **实盘验证**：真实账户运行两年，累计收益超 146 万元，大幅跑赢指数

## 适用场景
- 价值投资者对 A 股/港股/美股上市公司的快速初筛或深度研究
- 行业产业链全景扫描，寻找最佳投资标的
- 持仓异动时的快速新闻归因与论文重审
- 未上市公司的侦探式深度估值
- 财报季的原始报告精读，避免二手信息误导
- 个人投资者建立纪律化的投资研究流程，避免情绪化决策

## 同类对比
- **对比直接使用 ChatGPT/Claude**：通用问答缺乏结论强制性和反偏见流程，输出“一面之词”。AI Berkshire 提供固定的多视角框架、评分卡和决策纪律，可直接用于实盘。
- **对比专业金融终端（Bloomberg、Wind）**：后者提供数据和工具但不含投资哲学；AI Berkshire 将大师方法论编码为流程，免费开源，但依赖 Claude Code 的 API 费用。
- **对比其他 AI 投研工具**：多数商业产品为黑盒或仅输出“报告”，无开源流程可审计；AI Berkshire 裸露全部 Skill，允许用户修改和审计。
