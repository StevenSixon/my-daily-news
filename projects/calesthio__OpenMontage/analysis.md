## 它是什么
OpenMontage 是一个开源、Agent驱动的视频制作系统，可将 Claude Code、Cursor 等 AI 编码助手转化为完整的视频工作室。用户用自然语言描述需求，Agent 自动完成研究、脚本、素材生成、编辑和最终合成。它区别于普通“图片动画”工具，能通过免费档案资料检索真实运动镜头，剪辑成时间线并渲染成片。

## 为什么火
上线不久即获 17k+ Star，因它彻底简化了视频制作流程：无需专业技能，仅靠文字描述即可生成堪比专业水准的视频，且支持零 API 密钥的免费路径（Piper TTS + Archive.org 等）。示例视频从科幻预告片到吉卜力风格动画，成本低至 0.15 美元，极具吸引力。

## 技术栈
- 语言：Python 3.10+，Node.js 18+
- 视频合成：Remotion（React 渲染）或 HyperFrames（HTML/CSS/GSAP）
- 语音：Piper TTS（离线），ElevenLabs，OpenAI TTS
- 图像/视频生成：FLUX，Stable Diffusion，Google Veo，Kling 等（通过 fal.ai 网关）
- 媒体处理：FFmpeg
- 素材源：Pexels，Pixabay，Unsplash，Archive.org，NASA，Wikimedia Commons
- 编排：管道定义（pipeline_defs），技能注册，工具发现

## 核心能力
- 12 个预定义流水线，覆盖动画短片、产品广告、纪录片蒙太奇等多种类型
- 输入 YouTube 链接即可分析并生成类似风格新视频
- 多维度自我审查：ffprobe 校验、帧采样、音频分析、字幕检查
- 成本预估与控制，支持仅用 OpenAI 等单一 API 完成全流程
- GPU 本地视频生成支持（Wan2.1, Hunyuan 等）
- 代理能力完善，适合在 OpenClaw 等 agent 环境中自动运行

## 适用场景
- 内容创作者快速生成抖音/YouTube 短片
- 开发者将视频生成集成到自动化工作流
- 教育培训材料制作（解说动画）
- 产品宣传视频低成本打样
- AI 研究者探索 agent 在创意领域的应用

## 同类对比
同类开源视频生成项目多集中在单个模型调用，或仅能生成静态图片拼接动画。OpenMontage 的优势在于：
- 真正的“真实素材”剪辑能力，而非仅图片串烧
- 全流程自动化，从策划到渲染无需人工干预
- 灵活的提供商体系，自由组合收费/免费工具
- 与 AI 编码助手深度集成，而不是单独的命令行工具
缺点是目前对硬件（GPU）和 API 密钥仍有一定依赖，且复杂剧情的逻辑控制尚待观察。
