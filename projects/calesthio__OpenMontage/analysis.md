## 它是什么
OpenMontage 是一个开源的、由代理驱动的视频生产系统。它将 AI 编程助手（如 Claude Code、Cursor）扩展为一个完整的视频工作室。你只需用自然语言描述需求，系统便会自动执行主题调研、脚本撰写、资产生成（AI 图像或真实视频片段）、编辑与最终合成。它支持基于图像动画的视频和真实视频素材的检索与混剪，成本极低，并强调这是真正的“视频”制作，而非简单的静态图像动画化。

## 为什么火
- **极致自动化**：一句话即可生成完整视频，消除手动剪辑和素材搜集的繁琐。
- **极低成本**：利用免费素材和本地 TTS（如 Piper），一支风格化视频最低只需 $0.15。
- **代理友好**：内置 12 条管道、52 个工具和 500+ 代理技能，AI 代理能自主规划并执行多步骤视频制作。
- **真实视频能力**：不仅能生成 AI 图像动画，还可从免费媒体库和档案中检索真实动态素材并进行剪辑，区别于多数同类工具。
- **开源可扩展**：AGPL-3.0 许可，支持多种 API 供应商和本地 GPU 视频生成，社区可自由定制。

## 技术栈
- **核心语言**：Python 管程流程，Node.js 用于 Remotion 渲染。
- **渲染引擎**：Remotion（React 动画引擎）和 HyperFrames（HTML/CSS/GSAP）。
- **媒体处理**：FFmpeg。
- **语音合成**：Piper（本地离线）、ElevenLabs、OpenAI、Google 等多供应商支持。
- **图像/视频生成**：FLUX、DALL·E 3、Google Veo、Kling、Runway 等，可通过 fal.ai 或本地 GPU 接入。
- **免费媒体素材**：Pexels、Pixabay、Unsplash、Archive.org、NASA、Wikimedia Commons 等。
- **代理交互**：通过读取项目文件并执行命令，与主流 AI 编程助手（Claude Code、Cursor、Copilot、Windsurf、Codex）协作。

## 核心能力
- **管道化生产**：12 条预定义管道覆盖不同视频风格，代理可根据需求选择管道，自动分析参考视频的节奏、风格、结构。
- **多模态资产**：AI 图像生成、文本转语音（带语音指导）、背景音乐自动搜寻与配对、词级字幕自动生成与烧入。
- **自动质检**：ffprobe 验证、帧采样、音量分析、字幕检查等全流程自检，确保视频质量。
- **真实视频剪辑**：从开源和免费媒体库中检索真实视频片段，结合 AI 生成内容，实现纪录片式的混剪。
- **零成本启动**：不添加任何 API key 即可产出视频，利用本地 TTS、免费素材和动画引擎。
- **本地 GPU 支持**：可选启用本地视频生成模型（如 Wan2.1、CogVideo），进一步降低成本。

## 适用场景
- 社交媒体内容（YouTube Shorts、TikTok）的批量生产与灵感快速生成。
- 产品宣传片、广告短片，无需专业剪辑团队。
- 纪录片式自动故事生成，利用开源素材和旁白。
- 教育讲解动画，结合文本转语音和图像动画。
- AI 代理实践展示：多步骤复杂任务的自动化执行。

## 同类对比
- **对比商业 AI 视频工具（如 Synthesia、Pictory）**：OpenMontage 完全开源、代理驱动、可本地运行，透明且成本极低，但需要一定的技术环境搭建。
- **对比创意生成工具（RunwayML、Pika）**：后者侧重单段生成，OpenMontage 提供端到端的制作流水线，包含素材检索、剪辑、字幕合成等全流程。
- **对比传统剪辑软件（Premiere、DaVinci）**：零手动操作，全由代理自动决策并执行。
