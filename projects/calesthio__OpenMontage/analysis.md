## 它是什么
OpenMontage 是一个开源的代理式视频生产系统，能将 AI 编程助手（Claude Code、Cursor 等）转变为完整的视频制作工作室。它通过 12 条预设管线、52 个工具和 500 多个代理技能，将自然语言描述转化为可渲染的视频项目，支持从真实素材剪辑到 AI 生成动画等多种风格。

## 为什么火
- **代理式工作流**：完全由代理驱动，自动处理研究、脚本、资产生成、编辑和合成，大幅降低手动操作。
- **免费友好**：无需 API 密钥即可用 Piper TTS、Archive.org 等免费工具制作视频。
- **真实视频能力**：不仅仅是图片幻灯片，能使用真实素材和运动剪辑。
- **成本披露**：示例视频成本低至 $0.15，透明度高。

## 技术栈
- 语言：Python
- 依赖：FFmpeg、Node.js、Remotion 或 HyperFrames 前端渲染
- 核心组件：管线定义 (`pipeline_defs/`)、技能系统 (`skills/`)、工具注册表 (`tools/tool_registry`)
- 集成提供商：OpenAI、ElevenLabs、FAL、Google 等多种 API，也支持本地 GPU 视频生成（Wan2.1 等）

## 核心能力
- 多管线支持：图像动画、真实素材纪录片、产品广告、角色动画等
- 智能分析：从参考视频提取风格、节奏并生成差异化方案
- 渲染引擎：Remotion（React）用于数据驱动动画；HyperFrames（HTML/GSAP）用于动效图形
- 自审系统：ffprobe 验证、帧抽样、音频分析等
- 无密钥模式：本地 TTS、开放档案素材、免费股票 API

## 适用场景
- 快速生成社交媒体短视频（TikTok、Reels）
- 制作低成本广告、产品演示
- 内容创作者需要快速将创意变为视频
- 开发者在 IDE 中集成视频生成能力

## 同类对比
- 相比 RunwayML 等商业平台：开源、可自托管、成本更低，但需要一定的技术栈
- 相比普通的 AI 视频脚本工具：代理式端到端生产，而非仅生成提示词或静态图片
- 相比 ComfyUI + AnimateDiff：提供了更高层的抽象和管线管理
