## 它是什么
Godcoder 是一个完全开源、本地优先的 AI 编码代理，以原生桌面应用形式运行。用户带上自己的 LLM 密钥，源代码直接从本机发送到模型供应商，无需经过任何中间云后端。它不仅能辅助编码，还能进入“Harness 模式”自主构建和优化自己的代理工具链，以及“CoWork 模式”学习驱动 Open Cowork 桌面应用，执行点击、输入、发邮件等人类操作任务。

## 为什么火
- **隐私零妥协**：代码永不离开本机，所有 API 请求直接从用户机器到模型提供方，没有数据锁定的风险。
- **自主进化能力**：Harness 模式让代理自行搭建沙箱、筛选改进方向、执行并评估修改，将成功经验持久化，实现工具链的自我优化。
- **全面的任务覆盖**：从问答、规划、编码到完全自主的 Harness/CoWork 模式，再到终端、文件浏览器和检查点恢复，功能齐全。
- **开源与本地扩展性**：支持 MCP 服务器、语音 API 和图感知代码搜索，可由社区无限扩展。

## 技术栈
- 代理核心：纯 Rust 编写（`crates/agent/`）
- 桌面壳：Tauri 2（Rust 后端） + React 前端
- 上下文引擎（可选）：Go 服务，基于 tree-sitter 语法解析、Qdrant 向量搜索、FalkorDB 调用图遍历和 BM25 词法搜索
- 自我优化桥梁：Python 桥接 ResearchSwarm 记忆存储

## 核心能力
- **多模式交互**：Ask / Plan / Coding / Freestyle / Harness / CoWork 六种模式
- **原地文件编辑**与 diff 查看、检查点回滚
- **内置终端与文件浏览器**
- **任意 LLM 支持**：OpenAI、Anthropic 或任何 OpenAI 兼容 API
- **MCP 服务器扩展**（支持 stdio、HTTP、SSE 传输）
- **语音 API 集成**（配置本地 TTS/STT）
- **图感知代码搜索**（需启动 Context Engine）
- **Harness 模式**：代理自动搭建工具链沙箱，执行 route→plan→execute→evaluate→log→optimize 循环，逐步增强自身能力
- **CoWork 模式**：训练代理驱动 Open Cowork 桌面应用，执行真实 GUI/OS 自动化任务，并验证截图

## 适用场景
- 极度重视代码隐私的开发者，不希望代码上传到第三方平台
- 需要高度自定义、可自我进化的 AI 辅助编码工作流的高级用户
- 探索 AI 代理自主构建工具链与桌面自动化可能性的研究人员
- 希望用单一工具完成编码辅助、知识检索、桌面自动化任务的极客

## 同类对比
与 GitHub Copilot、Cursor 等云托管工具相比，Godcoder 的最大优势是完全本地运行、数据不离开机器。与 Continue、Open Interpreter 等开源本地工具相比，它额外提供了自治的 Harness 和 CoWork 模式，让代理不仅能被动辅助，还能主动优化自身并执行桌面任务，且具有图感知的代码检索能力。
