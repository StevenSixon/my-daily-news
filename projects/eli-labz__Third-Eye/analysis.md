## 它是什么

Third Eye 是一个生产级 OSINT 平台，将飞行跟踪、海事动态、CCTV 网络、地震监测、冲突区域等16个情报层聚合到单一 GPU 加速的地图界面中。所有数据通过 WebGL 渲染，支持 60fps 交互，且无需 API 密钥即可运行。

## 为什么火

项目发布仅一天即获 508 星，因其开箱即用的完整 OSINT 工作台特性吸引了安全社区。自托管、零配置快速启动，同时内置侦察工具和可选智能分析模块，填补了开源地理空间情报聚合的空白，降低了分析师搭建数据管道的门槛。

## 技术栈

- 框架：Next.js 16 (App Router, Turbopack)
- 语言：TypeScript 5
- 地图引擎：MapLibre GL JS (WebGL)
- 动画：Framer Motion
- 图标库：Lucide React
- 部署：Vercel Edge Network，支持 Docker 自托管

## 核心能力

- **16种实时情报层**：航空、海事、CCTV、地震、火灾、新闻、天气、太空、网络、冲突、加密币、制裁、Telegram OSINT 等，全部可独立开关。
- **GPU 加速渲染**：MapLibre GL 直接操作 WebGL，在大量实体下仍保持 60fps。
- **内置 RECON 工具包**：端口扫描、DNS 查询、WHOIS、SSL 检查、IP 情报、CVE 漏洞扫描、加密钱包追踪与 OFAC 制裁交叉验证。
- **Telegram OSINT**：无 Bot API 即可抓取公开频道帖子，经地理编码后标绘在地图。
- **可选 Smart System 模块**：基于现有数据流的决策支持流程（对象检测、轨迹异常、报告摘要、行动建议），所有输出需人工审核，带完整审计轨迹。
- **零配置启动**：核心层全部使用公开、无密钥数据源，克隆即可运行。

## 适用场景

- 安全分析师进行实时态势感知与事件监控
- OSINT 调查人员快速聚合多源公开情报
- 记者与研究人员追踪冲突、灾害、异常事件
- 自托管环境下的内部情报中心

## 同类对比

与 AlienVault OTX、Maltego 等 OSINT 工具相比，Third Eye 更侧重“空间化展示”和“自托管零成本”的即用性，而非深度关联分析。它弥补了开源领域缺少高性能地理情报面板的不足，但缺乏高级关联图谱和 API 集成能力。
