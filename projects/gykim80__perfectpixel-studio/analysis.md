## 它是什么
PerfectPixel 是一款基于 Wails + Go + React 的桌面应用，将一行文本描述转化为一个基础角色，并自动生成 100+ 动作动画（行走、奔跑、攻击、魔法等）及完整的 8 方向精灵集，导出兼容游戏引擎的格式。

## 为什么火
目前 AI 图像生成工具很少能解决游戏精灵制作中的硬约束：帧数必须精确、角色外观帧间一致、背景全透明且锚点不抖动。PerfectPixel 通过确定性后处理管道（YCbCr 抠图、投影剖面 + DP 最优帧切割、质心对齐、共享调色板量化）大幅提升 AI 输出的可用性，为独立游戏开发者和快速原型提供了低成本、高质量方案。

## 技术栈
- 后端：Go 1.25+，负责 pipeline（prompt 构建、图像处理、质量评分与重试控制）
- 前端：React 18，负责交互与画廊展示
- 外壳：Wails v2，打包为本地桌面应用
- AI 提供方：Gemini、OpenRouter、fal.ai、BytePlus（可配置）
- 图像处理：YCbCr 色度抠图、Hermite smoothstep 柔边、连通域填充、DP 优化切分、alpha 加权质心对齐、中位切分调色板量化

## 核心能力
- **文本到角色**：一次描述生成基础角色，自动去背景
- **100+ 动作预设**：覆盖行走、待机、攻击、魔法、表情等
- **8 方向生成**：AI 生成 5 方向，3 个方向由镜像派生，节省 37.5% 成本
- **自纠正质量循环**：生成 → 抠图 → 分割 → 检查 → 纠正型再生，最多 3 轮，基于最佳候选评分，失败时注入精确修复提示
- **像素化处理**：共享调色板量化 + 像素网格吸附，将 AI 伪像素转为真正点阵
- **多引擎导出**：精灵表、manifest.json、Aseprite 兼容 JSON、逐帧 GIF/APNG/PNG
- **会话与画廊**：工作状态持久化，结果自动归档
- **多 AI 提供方**：支持 Gemini/OpenRouter/fal.ai/BytePlus，配置灵活

## 适用场景
- 独立游戏开发者快速生成角色精灵原型
- 游戏设计 jam 中快速产出动作丰富的角色
- 低预算或单人项目减少美术外包成本
- 需要 8 方向精灵的 2D 游戏（RPG、动作游戏）

## 同类对比
相比大多数 AI 精灵生成工具（如仅生成单张图片或简单序列），PerfectPixel 的核心优势在于确定性后处理：通过信号处理手段保证帧对齐、背景透明和身份一致，并提供引擎可直接导入的格式。与商业工具如 Aseprite + 手动绘制相比，大幅度降低时间成本；与纯 AI 生成 API 相比，解决了生成结果不可控的问题。
