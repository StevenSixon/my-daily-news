## 它是什么
一个逆向工程项目，将微软Copilot网页聊天转换为OpenAI兼容的RESTful API和Python库。用户使用自己的Microsoft/Google账号登录一次后，即可通过本地`http://localhost:8000`端点免费调用Copilot的大模型，并获得流式响应、多轮对话支持。

## 为什么火
它为开发者提供了零成本接入高级LLM的途径，特别适合个人项目、原型验证、学习探索。相比需要付费API key的OpenAI服务，该项目降低了试验门槛，且对OpenAI生态工具（如openai SDK）完全兼容。

## 技术栈
- Python 3.9+
- Playwright（浏览器自动化登录）
- FastAPI + uvicorn（服务器端）
- 会话持久化（cookies + token）
- 令牌桶限流

## 核心能力
- **零成本API**：基于已有Copilot账号，无每日限额（但建议适度使用）
- **OpenAI兼容**：`/v1/chat/completions`和`/v1/models`端点，官方SDK可直连
- **多轮对话**：通过`conversation_id`维持上下文
- **流式输出**：支持`stream=true`的参数
- **自动登录与刷新**：首次运行弹出浏览器完成登录，后续自动复用会话
- **速率与并发控制**：服务器串行化请求，内置令牌桶限制（默认12次/分），防止上游封禁
- **Docker支持**：可容器化部署

## 适用场景
- 个人开发者用作免费对话API进行原型测试
- 需要轻量对话能力的本地应用（如桌面助手、CLI工具）
- 学习OpenAI API接口而无需付费
- 在匿名Copilot被屏蔽的地区使用已登录账号访问

## 同类对比
与直接使用OpenAI API相比，主要优势是免费、无密钥管理；劣势在于不稳定（需维护会话、无法保证SLA）、无模型选择、单会话并发能力弱，且受上游微软策略影响。与ChatGPT逆向项目类似，但该项目专注于Copilot，登录更灵活（支持Google账号）。
