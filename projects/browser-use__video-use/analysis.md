## 它是什么
Video-Use 是一个基于 LLM 代理的视频自动剪辑工具。你把原始素材放到文件夹，在 Claude Code 等命令行代理里说“剪成发布视频”，它就会读取音频转录和可选视觉合成图，生成剪辑决策列表（EDL），渲染输出 final.mp4，并自我评估每个剪切点，确保无爆音、无视觉跳跃。

## 为什么火
仓库在 1 个月内获得 10k+ Star，痛点清晰：非编软件学习曲线陡、重复性剪辑（去除口头禅、加字幕）枯燥。Video-Use 以文本为第一操作界面，LLM 通过结构化转录（每词时间戳、说话人分离）进行精准剪切，无需逐帧浏览。每次会话记忆持久化，迭代效率高。

## 技术栈
- 运行环境：Python，需 ffmpeg 和 yt-dlp。
- LLM 代理：Claude Code、Codex、Hermes 等，要求 Shell 访问权限。
- 音频转写：ElevenLabs Scribe（需 API Key），输出含词级时间戳和说话人标记。
- 可视化：timeline_view 生成波形+胶片条+词标签的 PNG，用于决策点。
- 动画层：可调用 HyperFrames、Remotion、Manim 或 PIL，并发子代理生成。
- 输出：EDL → ffmpeg 渲染 → 自评估循环（最多 3 次修正）。
- 持久化：project.md 保存会话状态，下次继续使用。

## 核心能力
1. 智能剪切：自动识别填充词、静音间隙、无效片段。
2. 自动调色：可为每段应用预设或自定义 ffmpeg 链。
3. 音频平滑：所有切割点 30ms 淡入淡出，避免音爆。
4. 字幕生成：默认全大写两词块，可定制样式。
5. 动画叠加：并发子代理生成 Remotion/Manim 等动画。
6. 自评估渲染：输出前在切割点检查视觉/听觉缺陷。
7. 零预设内容：不假设视频类型，通用处理。

## 适用场景
- 播客/访谈剪辑：快速清除口头禅，保留关键发言。
- 教学视频制作：将多段录制拼接为带字幕的成品。
- 旅行/Vlog：自动生成带转场和音乐卡的短视频。
- 社交媒体剪辑：一键生成不同比例、字幕样式版本。

## 同类对比
与传统非编软件（Premiere、DaVinci）相比，Video-Use 无需 GUI 操作，通过自然语言一次定义风格即可批量产出。与 AI 剪辑工具（如 Descript）差异在于：它是开源的、本地优先，并且将剪辑过程完全代理化，开发者可自定义每一步逻辑，并可集成到自有流水线。
