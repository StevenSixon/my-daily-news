## 它是什么

Palmier Pro 是一个用 Swift 从零构建的 macOS 视频编辑器，特色是深度集成 AI 生成与代码助手：编辑器中可直接调用 Seedance、Kling 等模型生成视频/图片，同时内置 MCP 服务器，让 Claude Code、Cursor、Codex 等 AI 助手通过标准协议操控时间线。编辑器本身（不含生成式 AI）完全开源（GPLv3），生成模块闭源，采用订阅制。

## 为什么火

- Y Combinator S24 项目，苹果生态原生应用，近 9k Star，吸引了大量对 AI + 创作工具结合的开发者。
- 提出“AI 原生视频编辑器”概念：不再只是插件，而是让 AI 成为协作者，通过 MCP 将 Claude 等变成时间线操作手。
- 精准卡位 macOS 26（Tahoe）与 Apple Silicon，利用 Metal 加速视觉效果，展现新技术栈潜力。

## 技术栈

- Swift + AppKit/SwiftUI（推测）实现原生 UI 与高性能渲染。
- Metal 用于颜色分级与特效合成（v0.4.0 已集成）。
- MCP 服务器通过 HTTP 暴露在 127.0.0.1:19789，支持 Claude Code/Desktop、Cursor、Codex 等常用 AI 编码工具。
- 音频处理支持 PCM 共享读取器、多轨同步，支持 AIFF-C、FLAC 等格式导入。
- 生成式 AI 部分闭源，未披露具体协议，或为自研管线与第三方 API 的组合。

## 核心能力

- **完整视频编辑**：多轨时间线、剪辑、变换、关键帧、分辨率导出（含 2K、匹配时间线）。
- **生成式 AI**：在时间线内直接生成 Seedance、Kling、Nano Banana Pro 等模型的图像/视频素材。
- **智能代理协作**：MCP 服务器让外部 AI 助手可以读取、修改项目，实现自动化剪辑。
- **多语言支持**：15 种语言，包括阿拉伯语等 RTL 语言。
- **免费编辑器**：无登录即可使用基础编辑功能，MCP 服务也免费开放。

## 适用场景

- 视频创作者想要用代码或 AI 助手自动化重复性编辑（如批量加字幕、调色、剪辑片段）。
- 开发者试验“AI 代理控制 GUI 工具”的交互范式，利用 MCP 构建自定义剪辑流程。
- macOS 生态重度用户，追求原生性能与 Metal 加速的本地编辑体验。
- 对 AI 生成内容与手工编辑混合工作流感兴趣的早期尝鲜者。

## 同类对比

- 对比 Premiere Pro/After Effects：Palmier Pro 更轻量，原生集成 AI 代理，而非通过插件层交互，但功能成熟度远不及 Adobe。
- 对比 CapCut：CapCut 更偏向移动/桌面全流程一键式模板与 AI 特效，Palmier Pro 开放 MCP，给技术用户更大的自定义空间。
- 对比 DaVinci Resolve：DaVinci 有强大的 API 与 Fusion，但并非为 AI 助手原生设计；Palmier Pro 从底层就考虑 Agent 协作。
- 独特优势：唯一一个以 MCP 标准正式支持 Claude/Cursor 等工具的视频编辑器，形成“写代码即剪视频”的新路径。

## 版本动态

- 社区贡献活跃，多语言本地化持续更新，关注者通过 Discord/X 反馈，项目迭代较快。
- 现阶段仍处于早期，编辑器基础功能稳定，但 AI 能力依赖闭源模块，长期发展取决于订阅模式与开放生态的平衡。
