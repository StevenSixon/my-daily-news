## 它是什么
Palmier Pro 是一个从零用 Swift 构建的 macOS 视频编辑器，最大特点是为 AI 代理（Claude、Codex、Cursor）提供一等公民支持：通过内置的 MCP 服务器，AI 可以直接读取/操作时间线、生成内容、执行剪辑。编辑器本身免费且开源（GPLv3），生成式 AI 功能闭源但可选购。

## 为什么火
- YC S24 背景，社区热度高（4k+ stars，创建仅 2 个月）
- 抓住了 AI Agent 与专业工具集成的趋势，MCP 生态关键节点
- 将“AI 替你剪视频”从概念变成可落地的工作流，开发者社区对新范式好奇

## 技术栈
- 语言：Swift
- 平台：macOS 26 (Tahoe) + Apple Silicon 专属
- 集成协议：MCP over HTTP (127.0.0.1:19789/mcp)
- AI 生成模型：Seedance, Kling, Nano Banana Pro (闭源部分)
- 分发：DMG，无 App Store 版本

## 核心能力
- 原生视频编辑：多轨时间线、波纹插入、剪辑分割、关键帧调节、撤销等
- MCP 服务器：应用运行时暴露 MCP 端点，AI 助手可获取时间线状态、执行剪切、添加素材、基于转录文本的剪辑
- 内置聊天代理：可在应用内与助手问答，复制代理消息
- AI 生成视频/图像：在时间线内直接触发生成（需订阅）
- 文件夹导入，轻量遥测面包屑，崩溃修复等工程打磨

## 适用场景
- AI/智能代理开发者：将视频编辑作为 MCP 工具集成到自主工作流
- 内容创作者：用自然语言让 Claude/Cursor 帮助剪辑、描述时间轴、批量操作
- 产品 Demo：展示 AI + 专业工具的协同可能性

## 同类对比
- Adobe Premiere Pro：闭源重量级，无原生 AI 代理接口，Palmier 以 MCP 开放性和 Swift 轻量化为区分
- CapCut/Final Cut Pro：无 MCP 集成，AI 能力封闭；Palmier 让 AI 直接操控时间线数据
- 其他 AI 视频工具（Runway 等）：主要提供生成能力，缺少可编程的时间线编辑器和本地 MCP 服务器
