## 它是什么
Palmier Pro 是一个从零用 Swift 构建的 macOS 视频编辑器，定位 Premiere Pro 的开源现代替代品，并深度集成了 AI 生成能力与 MCP（Model Context Protocol）服务器。用户可以在传统时间线编辑的同时，调用 Seedance、Kling 等 SOTA 模型生成视频/图片，并允许 Claude Code、Cursor、Codex 等 AI agent 通过本地 HTTP 接口直接操作剪辑项目。

## 为什么火
1. **AI + 视频编辑的协议化**：MCP 已成为连接 LLM 与工具的事实标准，Palmier 是首个将完整视频编辑器能力暴露为 MCP 工具的开源项目，使得视频生成和编辑工作流可被任何 agent 编排。
2. **开源但好用**：基础编辑器完全开源且免费，无限制，这与 CapCut 等工具形成对比，同时原生 Swift 实现保证了 Apple Silicon 上的流畅体验。
3. **YC S24 背景**：有孵化器背书，社区活跃，Star 数快速增长。
4. **本地优先，隐私友好**：MCP 服务完全运行在本地，处理编排无需上传云端。

## 技术栈
- **语言与框架**：Swift 原生，专为 macOS 26 (Tahoe) + Apple Silicon 打造。
- **架构**：时间线引擎、渲染、导出均自主实现；MCP 服务器采用 HTTP 传输，绑定 `127.0.0.1:19789`。
- **集成模型**：Seedance, Kling, Nano Banana Pro（闭源生成部分）。
- **开放模型**：编辑器核心、MCP 服务器、agent 聊天窗口完全开源（GPLv3）。

## 核心能力
- **完整的 Swift 原生视频编辑**：旋转/裁剪叠加、速度调节、源素材变换、填充词处理等，并在 v0.3.4 中修复了旋转黑边等渲染问题。
- **MCP 工具集**：支持 `get_transcript`、`ripple_delete_ranges`、`set_clip_properties` 等工具，agent 可读写时间线。
- **多 agent 接入**：一行命令即可接入 Claude Code、Codex、Cursor，Claude Desktop 甚至提供一键安装扩展。
- **内置生成式 AI**：可在时间线上直接生成视频或图片内容（需订阅）。

## 适用场景
- 内容创作者想用自然语言批量剪辑或删除填充词，提高效率。
- 开发者构建自动化视频流水线（例如基于转录删除静音或填充词）。
- AI 研究者实验视频领域的 tool-use 范式。
- 任何想在本地保持完整控制，同时享受 AI 辅助剪辑的 macOS 用户。

## 同类对比
- **Premiere Pro / DaVinci Resolve**：功能更全但闭源、无 API 让 agent 直接操控，且需付费。
- **CapCut**：免费但有水印/限制，无法被外部程序编排。
- **其他视频编辑 SDK**（如 Remotion）：偏向程序化生成，而非交互式编辑。Palmier 融合了交互式 UI 与程序化控制，对两类用户都友好。
- **独有优势**：MCP 原生集成，使所有符合 MCP 的 agent 都可成为你的剪辑助手，且基础功能永远免费。

## 版本动态
